# go-kinspector

A CUI tool inspect your AWS kinesis streams, check records, and put new records.

![screenshot](screenshot.png)

## Build

```zsh
go build .
```

## Install

### With Go

```zsh
go get github.com/borgmon/go-kinesis-inspector
go-kinesis-inspector
```

### Download binary

```zsh
./go-kinesis-inspector
```
